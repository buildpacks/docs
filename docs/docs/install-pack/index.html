Installing pack

You can install the most recent version of pack (version 0.0.9) as an executable binary on the following operating systems:

  • macOS
  • Linux
  • Windows

pack requires Docker to run

Install Docker Community Edition

macOS

To install pack on macOS, the easiest way is to use Homebrew:

$ brew tap buildpack/tap
$ brew install pack

To install manually instead, fetch and unpack the tarball:

$ wget https://github.com/buildpack/pack/releases/download/v0.0.9/pack-0.0.9-macos.tar.gz
$ tar xvf pack-0.0.9-macos.tar.gz
$ rm pack-0.0.9-macos.tar.gz
$ ./pack --help

From there, you can copy the executable to a directory like /usr/local/bin or add the current directory to your PATH.

Linux

$ wget https://github.com/buildpack/pack/releases/download/v0.0.9/pack-0.0.9-linux.tar.gz
$ tar xvf pack-0.0.9-linux.tar.gz
$ rm pack-0.0.9-macos.tar.gz
$ ./pack --help

From there, you can copy the executable to a directory like /usr/local/bin or add the current directory to your PATH.

Windows

You can install the Windows executable for pack by downloading the Windows ZIP file.
